# 快速开始指南

## 🎯 5分钟快速上手

### 第一步：安装插件

**Windows 用户：**

1. 下载 `currency-exchange-rates-1.0.0.vsix` 文件到桌面
2. 打开 VS Code
3. 按键盘上的 `Ctrl + Shift + P`
4. 在弹出的输入框中输入：`install vsix`
5. 选择 `Extensions: Install from VSIX...`
6. 找到并选择桌面上的 `.vsix` 文件
7. 等待几秒钟，看到"安装成功"提示
8. 点击右下角的"重新加载"按钮

✅ 安装完成！

### 第二步：查看汇率

安装成功后，你会在 VS Code 底部右侧看到：

```
💱 USD/CNY: 7.2345 ↑
```

这就是实时汇率显示！

**点击这个汇率信息**，会打开一个新页面，显示所有主要货币的汇率。

### 第三步：使用计算器

想知道 100 美元等于多少人民币？

1. 按 `Ctrl + Shift + P`
2. 输入：`汇率计算`
3. 在打开的计算器中：
   - 输入金额：`100`
   - 选择"从"：`USD`（美元）
   - 选择"到"：`CNY`（人民币）
   - 点击"转换"按钮

立即得到结果！

### 第四步：自定义设置

想在状态栏显示欧元对美元的汇率？

1. 按 `Ctrl + ,` 打开设置
2. 在搜索框输入：`currency exchange`
3. 找到"Status Bar Currency"
4. 从下拉菜单选择：`EUR/USD`

状态栏立即更新！

## 🎨 功能一览

### 1️⃣ 状态栏显示

- **位置**：VS Code 底部右侧
- **显示内容**：货币对 + 汇率 + 涨跌箭头
- **点击操作**：打开详细汇率页面
- **自动更新**：每5分钟刷新一次

### 2️⃣ 实时汇率查看

**打开方式：**
- 点击状态栏的汇率信息
- 或按 `Ctrl+Shift+P`，输入"查看实时汇率"

**功能：**
- 显示18种主要货币的实时汇率
- 双向汇率显示（如：CNY→USD 和 USD→CNY）
- 一键刷新按钮
- 显示更新时间

### 3️⃣ 汇率计算器

**打开方式：**
- 按 `Ctrl+Shift+P`，输入"汇率计算器"

**功能：**
- 输入任意金额
- 选择源货币和目标货币
- 一键转换
- 支持货币互换（点击 ⇄ 按钮）
- 显示详细计算结果

### 4️⃣ 手动刷新

**操作方式：**
- 按 `Ctrl+Shift+P`，输入"刷新汇率"

**效果：**
- 立即获取最新汇率数据
- 清除缓存
- 显示刷新成功提示

## ⚙️ 常用设置

### 更改状态栏显示的货币对

```
设置 → 搜索 "currency exchange" → Status Bar Currency
```

**热门选择：**
- `USD/CNY` - 美元/人民币（默认）
- `EUR/CNY` - 欧元/人民币
- `GBP/CNY` - 英镑/人民币
- `HKD/CNY` - 港币/人民币
- `EUR/USD` - 欧元/美元

### 调整刷新频率

```
设置 → 搜索 "currency exchange" → Refresh Interval
```

- 默认：300秒（5分钟）
- 最小：60秒（1分钟）
- 最大：3600秒（1小时）

### 关闭涨跌指示

```
设置 → 搜索 "currency exchange" → Show Change Indicator
```

取消勾选即可隐藏涨跌箭头。

## 💡 使用技巧

### 技巧1：快速查看多个货币

在"实时汇率"页面，你可以同时看到所有主要货币的汇率，无需切换。

### 技巧2：计算器支持小数

在计算器中可以输入小数金额，如 `99.99`。

### 技巧3：货币快速互换

在计算器中，点击中间的 `⇄` 按钮可以快速交换源货币和目标货币。

### 技巧4：回车键快速转换

在计算器的金额输入框中，按回车键可以直接执行转换，无需点击按钮。

### 技巧5：查看历史变化

状态栏的箭头指示器会显示汇率变化：
- `↑` 表示汇率上涨
- `↓` 表示汇率下跌

## ❓ 常见问题

### Q: 状态栏不显示汇率？

**A:** 检查以下几点：
1. 确认插件已安装并启用
2. 检查网络连接
3. 尝试手动刷新（`Ctrl+Shift+P` → "刷新汇率"）
4. 重启 VS Code

### Q: 汇率数据准确吗？

**A:** 数据来自可靠的第三方汇率 API，每5分钟更新一次。但请注意：
- 仅供参考，不作为交易依据
- 可能存在几分钟的延迟
- 与银行实际汇率可能有差异

### Q: 可以添加更多货币吗？

**A:** 当前版本支持18种主要货币。如需更多货币支持，请联系开发者。

### Q: 插件会影响 VS Code 性能吗？

**A:** 不会。插件：
- 仅在后台定时获取数据
- 使用缓存机制减少网络请求
- 资源占用极小

### Q: 需要注册或付费吗？

**A:** 完全免费，无需注册，开箱即用！
