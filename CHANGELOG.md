# 更新日志

## [2.0.0] - 2026-01-12

### 新增功能
- ✨ **历史汇率查看功能**
  - 支持查看货币对的历史汇率走势
  - 提供7天、30天、90天、180天、1年等多个时间范围选择
  - 展示汇率走势图表，直观显示汇率变化趋势
  - 显示最高、最低、平均汇率等统计数据
  - 支持货币对一键对换，快速查看反向汇率
  - 数据来源：Frankfurter API

### 使用方式
- 在侧边栏点击汇率卡片即可查看该货币对的历史汇率
- 在实时汇率界面点击表格行也可查看历史汇率
- 在历史汇率界面可以选择不同的时间范围
- 点击"对换"按钮可以查看反向货币对的历史汇率
